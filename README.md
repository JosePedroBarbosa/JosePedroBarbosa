## Featured Project

### PoliTutor — Socratic AI Tutor

**Final degree project developed at ESTG**

PoliTutor is a Socratic AI tutor designed to support programming students in higher education. Instead of providing direct solutions, it guides students through questions and contextual hints grounded in real course materials.

The system was built around a RAG architecture, reinforced with custom guardrails, and validated through benchmarks covering both retrieval and response generation.

**Key engineering areas:**

- Retrieval-Augmented Generation using course materials
- Socratic response generation
- Custom orchestration and guardrail logic
- Adversarial guardrail testing
- Benchmark-based retrieval and generation evaluation

**Tech:** LangChain, ChromaDB, FastApi, React, Python
